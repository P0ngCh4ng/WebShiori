# web shiori
webしおりは、ウェブページの読んだところまでを保存することができる拡張機能です。

web shiori is a a extention for chrome, used for save the point where you read.

# Features
保存したいページ上でクリックを行うと、その地点までの読んだところを保存することができます。

ブックマークに保存したり、他の拡張機能で一時保存した場合でも、もう一度そこから開くことができます。

If you click on webpage, this extention will save the point where you read.

Even you close and save to bookmark the webpage, you can open.
